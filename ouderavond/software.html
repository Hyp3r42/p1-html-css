Software



Info:

In software zie je welke programma's en apps moet gebruiken voor je opdrachten.

Die programma's en apps zijn office 365, Trello, Visual studio code en github.

Microsoft Office 365: Office 365 is een suite van kantoortoepassingen aangeboden door Microsoft. Het omvat programma's zoals Word, Excel, PowerPoint, en meer. Hoewel Office 365 meestal een betaalde service is, biedt Microsoft de online versie van deze tools gratis aan via Office.com, met beperktere functionaliteit in vergelijking met de betaalde versie.

Trello: Trello is een projectmanagementtool die gebruikmaakt van visuele borden en kaarten om taken en projecten te organiseren. Trello biedt een gratis versie met beperkte mogelijkheden, zoals het aantal borden en kaarten dat je kunt maken, maar het kan nog steeds effectief zijn voor kleine teams en persoonlijk gebruik.

Visual Studio Code: Visual Studio Code, vaak afgekort als VS Code, is een populaire en krachtige code-editor ontwikkeld door Microsoft. Het is gratis en open-source, en het biedt ondersteuning voor verschillende programmeertalen en een scala aan extensies die door de gemeenschap zijn gemaakt.

GitHub: GitHub is een platform voor het beheren van softwareontwikkelingsprojecten en het delen van code. Terwijl GitHub verschillende prijsniveaus heeft, waaronder betaalde opties, biedt het ook gratis repositories voor open-sourceprojecten. Dit betekent dat je gratis openbare code kunt hosten en samenwerken met andere ontwikkelaars.
